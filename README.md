# Holberton Core Engineering

Module d'ingénierie logicielle de Holberton School, centré sur Python. Le dossier regroupe les fondamentaux du langage jusqu'à la programmation orientée objet.

## Projets

| Dossier | Contenu |
|---|---|
| `python_fundamentals` | Bases de Python : variables, boucles, fonctions, structures de données, modules |
| `python_advanced` | Notions avancées : compréhensions, fonctions de première classe, gestion d'erreurs, tests |
| `python_oop` | Programmation orientée objet : classes, héritage, encapsulation, méthodes spéciales |

## Prérequis

- Python 3.8 ou plus récent
- Style PEP 8 (pycodestyle)

## Utilisation

```bash
cd python_fundamentals
python3 nom_du_script.py
```
